# mk5
HTML 5 UI Framework

No dependencies whatsoever!


run with:
```
python -m http.server 8000
```


## Based on Web Components ##

Makes heavy use of Web Components. Those consist of three parts:
- Custom Elements (JS)
- HTML Template (HTML)
- Shadow DOM (CSS)
[Ref](https://developer.mozilla.org/en-US/docs/Web/Web_Components)
